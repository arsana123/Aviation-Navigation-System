A web-based aviation navigation system that helps visualize flight paths, locations, and navigation data using interactive maps and real-time inputs.

🚀 Features
🌍 Interactive map integration using Google Maps API
✈️ Flight path visualization
📍 Location tracking and navigation
🧭 User-friendly interface
🔐 Backend support using PHP
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: PHP
API: Google Maps API
Server: XAMPP (Apache)
📂 Project Structure
Aviation-Navigation-System/
│── index.php
│── css/
│── js/
│── assets/
│── includes/
▶️ How to Run Locally
Install XAMPP

Move project to:

C:\xampp\htdocs\
Start Apache from XAMPP

Open browser and go to:

http://localhost/Aviation-Navigation-System/
🔑 API Configuration

Add your Google Maps API key in the JavaScript file:

const API_KEY = "YOUR_API_KEY";
Enable required APIs in Google Cloud Console
